炸了一个又一个手机之后，三星电子终于宣布暂停生产Note 7

发稿时间：2016/9/12 10:02

 

就在今天（10月10号），来自韩联社报道称，三星电子合作伙伴官员表示，三星电子已经暂停了Galaxy Note 7的生产。

韩联社引述三星某供应商的一位未具名管理人士称，该公司暂时停止了Galaxy Note 7智能手机的生产。该人士称，这一决定可能包括负责该产品全球供应的越南工厂。还称，作出这一决定是出于对全球消费者安全的考虑。

目前尚不清楚公司将何时恢复该机型的销售，但三星官方向新闻机构表示，公司已经与中、美、韩三国安全监管部门合作实施该措施。

这样一场电池爆炸的闹剧，已经漫长的持续了2个月之久。

早在8月2号，三星在美国召开新闻发布会，发布了一款新手机Note 7 。Note系列从5越过6直接到了7，估计谁都明白三星这样跳级，意味着什么。当大家把视线都放在Note 7的各个数据上，想了一系列三星与苹果对比的话题时，Note 7的3500毫安的电池却陆陆续续出现了安全隐患。

就在发布会后不到一月的时间里，就有Note 7疑似发生爆炸的事件，出现在了韩国国内。而与此同时，三星也决定推迟Galaxy Note 7出货，并在一份声明中称：“Galaxy Note 7的出货被推迟，因为我们要对产品质量进行额外的测试。”

网络上有一句流传颇广的话：韩国人一生逃不开的三件事：死亡、税收和三星。而讽刺的是，这一次，三个事件同时出现在了一款产品上。而且，这也仅仅是三星Note7电池爆炸事件的开始，随着时间的推移，爆炸事件层出不穷。

在大洋的彼岸，美国历来是三星手机的最为重要的市场之一，继韩国的用户反映手机电池出现问题后，在美国销售的Note 7 也陆陆续续出现了爆炸事件。

上市仅一个多月，在全球范围内已经发生了35起因锂电池质量缺陷造成起火爆炸的事故。于是，三星决定召回两周前才上市的250万支Note 7手机，据彭博估计，召回成本可能多达10亿美元。在与新iPhone正面交锋的节点上，三星遭受的损失不仅仅是10亿美元所能量化的。

然而，Note 7的噩梦并没有随着三星手机的召回所停止。

在接近9月中旬的时日里，多国城市交通管理部门，心照不宣的“封杀”三星Note 7：

已有包括日本国土交通省、欧洲航空安全监管机构（EASA）、加拿大交通部、印度民航总局（DGCA）、越南航空局等各国交通管理部门。

除此之外，澳洲航空、捷星航空、维珍澳大利亚航空、新加坡航空、美国航空、达美航空、联合航空、亚航、阿联酋航空、巴基斯坦国际航空公司等多家境外航空公司也先后对该款手机发布警告、禁令。

9月9日、10日，台湾地区华航、长荣、复兴、远东、台湾虎航等公司和香港地区国泰、港龙、香港航空相继宣布对三星Galaxy Note7手机的禁令。

国内的航空公司，民航局和春秋航空先后发布安全预警，重申旅客应严格遵守中国民航局安全规定，飞行全程中，严禁使用包括三星Galaxy Note 7在内的任何手机和充电，或作为托运行李运输。海航、首都航等航空公司也已明确禁止旅客携带三星Note 7乘机。

但是令人惊讶又不解的一幕却发生了。

中国市场并没有在在三星手机的召回名单中，三星方面申明：

“我们再次重申，自9月1日起在中国市场发售的Galaxy Note7国行版本，由于采用了不同的电池供应商，而不在此次更换范畴，可放心购买及使用。”

但打脸的是，不久之后就有了三星Note 7国行版发生爆炸。29日下午，三星电子在其官网向中国消费者致歉，并称三星“从未且永远不会对中国采用双重标准”，并承诺针对每起事件”都会在第一时间主动联系用户并获取相关信息”。

但即使是更换后的手机，Note 7 依然出现了问题。在10月5日，在美国西北航空一架飞机起飞前，一部Galaxy Note 7 手机起火，机上乘客不得不进行疏散。据报道，该手机已经是三星已维修过的、可安全使用的手机。

而美国的电信公司 AT&T 在周末宣布，将不再销售或为用户更换新Note 7，前来换货的用户可以更换为其他手机。

“基于近期的事故报告，现在我们不再为用户更换新的Galaxy Note 7，以等待这些事故得到进一步调查，”AT&T发言人弗莱彻·库克(Fletcher Cook)周日在一份电邮声明中称，“我们也鼓励使用被召回Galaxy Note 7的用户到AT&T商店更换另外一部三星智能机或其它智能机。”

失去运营商渠道，这一消息对于三星无疑是致命的。失去了 AT&T 的信任，也可能正是三星宣布停产的导火索。

在被人们熟知的消费电子领域之外，三星在韩国早已成长为一个覆盖机械、化工、金融、保险（放心保）等领域的庞大商业帝国。但是，最为推动下半年手机销量的主力机Galaxy Note 7，如今却只能暂停生产，让人对于三星接下来的日子，不得不产生悲观的情绪。

当然，更大的问题摆在了三星的面前：如何重新获取运营商和用户的信任。
